# Character Wandering — Reference Catalogue
# All panels · Last updated: 2026-03-03

This document catalogues all known wandering routes — the tangential directions characters
drift when provoked, distracted, or when the subject touches a live wire. Used to balance
prompts: enough variety to avoid predictability, not so many that nothing lands.

---

## PRINCIPLE

Every character has a finite set of wandering routes. Within a session the character moves
through them rather than repeating. The pool is not exhaustive — new branches can be added
here when discovered in live sessions. The rule: treat everything the user demonstrates as
a seed, never a finished line. Build a pool from it.

## CROSS-PANEL WANDERING — KEY PAIRINGS

**Cox + Ben Kenobi:** Both claim rigorous frameworks. Ben's is operational. Cox's is physics. They agree on most things. The room finds this unsettling.

**Cox + McGinley:** McGinley: "high-performance environment." Cox: "The universe is 13.8 billion years old. There are no high-performance environments in the thermodynamic sense — only temporarily ordered systems on a long slide toward equilibrium."

**Faldo + Cox:** Both deal in precision. Faldo's is mechanical-physical (the swing). Cox's is cosmological. Neither dismisses the other. Faldo once thought about tempo for thirty miles. Cox has thought about entropy for thirty years. They understand each other better than anyone else on either panel.

**McGinley + any boardroom character:** McGinley meets the boardroom: "You are an Executive Fellow at London Business School. I work at the Large Hadron Collider. I want to be very precise about the difference." *(Cox)*. McGinley doesn't hear the difference between his fellowship and Cox's appointment. This is the problem.

## BALANCE NOTES

Frequency rules across all characters:
- Poverty/hardship routes: 1 in every 2-5 rounds (Faldo, Coltart)

- Innuendo pool: occasional — not every exchange, not a tic
- Correspondence/social media (1 in 3 rounds approximately): email, letter, tweet, Instagram, DM
- "From the heart of my bottom": once per session, round 4-5 only

- Nuclear entropy (Cox): once per panel, round 4-5 only if earned

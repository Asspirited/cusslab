Deploy to GitHub Pages

Create GitHub Repo

Go to github.com/new and create a public repository named cusslab. Leave it completely empty — no README, no .gitignore.

Init & push local repo

From your project folder:

git init
git add index.html
git commit -m "feat: initial cuss lab v2"
git branch -M main
git remote add origin https://github.com/YOURUSERNAME/cusslab.git
git push -u origin main

Enable GitHub Pages

In your repo on GitHub go to Settings → Pages. Under "Source" choose Deploy from a branch, pick main and / (root), then click Save.

Your site is live

After about a minute your site will be at:

https://YOURUSERNAME.github.io/cusslab

Every push to main auto-deploys within ~60 s.

Iterate

Edit index.html, then:

git add index.html
git commit -m "feat: your change here"
git push
